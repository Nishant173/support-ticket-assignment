npm init -y
npm i -D webpack webpack-cli
npm i -D @babel/core babel-loader @babel/preset-env @babel/preset-react babel-plugin-transform-class-properties
npm i react react-dom prop-types
Create `.babelrc` file in root directory, with "presets" and "plugins"
Create `webpack.config.js` file in root directory, with `module.exports` object
Change the object at `package.json`. Alter the `scripts` key as follows
```
"scripts": {
    "dev": "webpack --mode development ./ticketter/frontend/src/index.js --output-path ./ticketter/frontend/static/frontend/main.js"
}
```
